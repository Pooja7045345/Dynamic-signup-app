## Getting Started

## install

### npm

```
npm install
or
npm install --legacy-peer-deps
```

### yarn

```
yarn install
```

## 2.Start

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## 3. Build

```sh
npm run build or yarn build
```
